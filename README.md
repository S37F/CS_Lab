### Prerequisites

### Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/S37F/CS_Lab.git
   cd CS_Lab
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
